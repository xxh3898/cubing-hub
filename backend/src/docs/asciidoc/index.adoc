== 개요

Cubing Hub 백엔드 REST API 문서입니다.

== 공통 에러 응답

=== 응답 구조

|===
| 필드 | 타입 | 설명

| `status`
| Number
| HTTP 상태 코드

| `message`
| String
| 실패 원인 또는 안내 메시지

| `data`
| Null
| 실패 응답에서는 `null`
|===
